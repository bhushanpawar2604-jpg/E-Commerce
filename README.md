# 🛒 MegaMall - Flutter E-Commerce App

MegaMall is a modern Flutter E-Commerce application built using Provider State Management and Clean Folder Structure.

The application provides a complete shopping experience including product browsing, search, wishlist, cart management, checkout flow, order tracking, authentication screens, and profile management.

---

## 🚀 Features

### Authentication
- Login Screen
- Register Screen
- OTP Verification
- Forgot Password
- Reset Password

### Home
- Banner Slider
- Featured Products
- Product Grid
- Bottom Navigation

### Products
- Product Details
- Product Reviews
- Product Search
- Product Categories
- Product Filtering
- Product Sorting

### Wishlist
- Add to Wishlist
- Remove from Wishlist

### Cart
- Add to Cart
- Remove from Cart
- Quantity Management
- Cart Total Calculation

### Checkout
- Address Screen
- Payment Screen
- Order Success Screen

### Orders
- Order History
- Order Tracking UI

### Profile
- User Profile
- My Orders
- Wishlist Overview
- Logout Functionality

---
